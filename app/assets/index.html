Grid

Base Layout system. contains a base 5 row and 9 column grid
the grid accepts panels that works as logical separator between content
the panels contain other cards with content.

Classes
Sq-grid-main-desktop: - define a 5 row, 9 col grid
Sq-grid-gap: - Adds a 1em gap betwen rows and columns
Sq-grid-gap-row: - Adds a 1em gap betwen rows
Sq-grid-gap-column: - Adds a 1em gap betwen columns
            
                <div class="sq-grid-main-desktop sq-grid-gap">
                    ...Add panels here...
                </div>

Panels

Panels are placed on a grid as a logical divider between different types of content. all panels content are adjust by use of flex classes

Classes
sq-panel-Base panel class with all the common functionality.
sq-panel-main-top - Creates a panel that spans the 2 first rows
sq-panel-main-bottom- Creates apanel that spans three bottom rows
sq-panel-main- creates a panel that spans the entire grid
main
top overlays main panel by being rendered after use z index to change
bottom overlays main panel by being rendered after use z index to chang

sq-background - This variable is using the background color.
sq-background-secondary - This variable is using the secondary background color.
sq-black - This variable is using the black text color.
sq-grey - This variable is using the grey text color.
